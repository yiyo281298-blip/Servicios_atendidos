Inspección de Policía

Modelo Predictivo de Servicios Atendidos

Utilizando Machine Learning para optimizar los recursos y mejorar el tiempo de respuesta.

Conocer el Proyecto

El Problema y Objetivo

Las inspecciones de policía reciben diariamente cientos de solicitudes, que van desde querellas hasta medidas de protección y audiencias de conciliación. El desafío principal radica en la gestión de recursos de manera eficiente.

El objetivo: A través de un modelo de Regresión Lineal implementado bajo la metodología CRISP-ML, predecimos la cantidad de servicios requeridos según variables temporales, logrando así anticiparnos a la demanda y prepararnos de forma proactiva.

Metodología CRISP-ML

Tabla de Variables

El modelo utiliza las siguientes características para realizar sus estimaciones.

Variable Tipo Descripción
AÑO Categórica Año en el que se prestó el servicio (ej. 2020, 2024).
MES Categórica Periodo del año (ej. Enero a Diciembre, Trimestres).
TIPO DE SERVICIO Categórica Clasificación del servicio (Querellas, Citaciones, etc.).
CANTIDAD Numérica (Target) Variable a predecir. Volumen total del servicio.
